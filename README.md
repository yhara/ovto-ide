# ovto-ide

Example of [Ovto](https://yhara.github.io/ovto) middleware.

## Features

- Show app state
- REPL window (movable/resizable)

## How to run

```
$ bundle install
$ bundle exec rackup
$ open http://localhost:9292/
```

## Files

- ovto/ - Contains Ovto app (= client side code)
  - app.rb
- app.rb - Sinatra app (= server side code)
